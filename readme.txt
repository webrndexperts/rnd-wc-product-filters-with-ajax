== Description ==

Woocommerce Ajax Product Filter with that when you install this plugin that will auto change and load the product wihtout loading or refreshing the page. This use the Ajax to load the page.
WIth this plugin you can do following
- Load the product with ajax on shop page
- Load the product with ajax on Category page
- Filters like Sorting of products, Pricing, reviews etc will work with ajax
- Show Product based on category with Ajax
- Ajax pagination for products

=== Privacy Notes ===

This plugin does not track any additional data other than what WordPress natively collects upon registration. It just makes the data visble.

= GDPR (General Data Protection Regulation) =

The plugin is GDPR compliant. It does not use or store any kind of user information/data. In fact, it's functionality has nothing to do with user data (personal or otherwise). So there's that.

== Installation ==

- Only you need to install the wordpress.
- Either using FTP or Dashboard you can install the plugin.
- Once you install this plugin and activate then this plugin will auto chagne the shop and category page with ajax
